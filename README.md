How to run

1. Build code:

    $ ```make build```

2. Run rmi registry:

    $ ```make registry```

3. Run two servers (e.g separate terminals)

    $ ```make host name=hostname1```

    $ ```make host name=hostname2```

4. Run client in separate terminal (it will execute our functionality)

    $ ```make run name1=hostname1 name2=hostname2```
